Website Incrível
Sistema que auxilia a checagem de aprendizado

Nosso Produto
O Moodle Website Incrível foi desenvolvido pensando em facilitar a vida de professores e alunos e de empresas que desejam implementar atividade para diferentes públicos por meio de um site que possui um layoult retrô incrível e que também exige uma baixa curva de aprendizado para todos os usuários.

Moodle Website Incrivel
O Website Incrível é um sistema de gerenciamento de aprendizado baseado na web, usado para intermediar a conexão entre professor e aluno. O Moodle é usado em todo o mundo para ensinar on-line, com mais de 100 milhões de usuários em mais de 200 países.

Cadastro
Todo novo visitante do site do site pode se cadastrar para ter acesso a todas as funcionalidades do site, inicialmente o visitante pode selecionar a opção 'Cadastrar', informações como email, senha, nome, idade e CPF são exigidas, e nossa plataforma também permite o usuário selecionar um tipo de perfil (perfil tipo aluno ou perfil do tipo professor).

Login
Após o cadastro, o usuário pode fazer login no site, para isso ele deve informar o email e a senha cadastrados, e então será redirecionado para a página inicial do site.

Professor: Criação de atividades/provas
O professor pode criar atividades e provas para seus alunos, para isso ele deve selecionar a opção 'Criar atividade', e então informar o nome da atividade, a data de entrega, o tipo de atividade (atividade ou prova) e o conteúdo da atividade. Nessa primeira versão da plataforma, é possível criar uma prova/atividade com 10 questões abertas (que exigem respostas dissertativas).

Após selecionar a opção para criação de prova/atividade o professor é então redirecionado para a página de criação, e então ele deve escrever o enunciado de cada uma daz 10 questões. Após isso, ao clicar em submeter atividade a prova/atividade é acrescida no banco de dados, de modo que os alunos podem acessá-la e responder as questões.

Aluno: realização de provas/atividadeds
O aluno pode responder as atividades/provas criadas pelo professor, para isso ele deve selecionar a opção 'Responder atividade', e então selecionar a atividade/prova que deseja responder. Após o aluno responder as 10 questões com enunciados definidos pelo professor, a atividade/prova pode então ser submetida pelo aluno de modo que as respostas são subidas em um banco de dados.

Professor: correção de atividades/provas
O professor pode corrigir as atividades/provas criadas por ele, para isso ele deve selecionar a opção 'Corrigir atividade', e então selecionar a atividade/prova que deseja corrigir, de modo que cada envio por cada aluno. fica disponível na página de correção de provas do professor, então após o professor selecionar a atividade/prova ele é redirecionado para a página de correção, onde ele deve então corrigir as 10 questões abertas da atividade/prova enviadas pelo aluno. Após isso, ao clicar em submeter correção da prova/atividade, é atualizada no banco de dados, de modo que o aluno pode verificar a nota correspondente à prova/atividade realizada. Um detalhe importante é que a cada nova correção que é feita pelo professor, a atividade/prova corrigida deixa de aparecer na página do professor, de modo que facilita a sua organização.

Aluno: visualização de notas
O aluno pode visualizar as notas das atividades/provas realizadas por ele, para isso ele deve selecionar a opção 'Visualizar atividades', e então ele poderá visualizar a nota das atividades submetidas e corrigidas pelo professor. Após o aluno selecionar a atividade/prova ele é redirecionado para a página de visualização de notas, onde ele pode ver a nota correspondente à prova/atividade realizada por ele.

Outros detalhes sobre o nosso Moodle Website Incrível
O Moodle Website Incrível foi desenvolvido para facilitar a vida de alunos e professores. Pensando nisso, os desenvolvedores criaram uma feature que permite que diferentes professores possam corrigir provas de outros professores, de modo que o professor responsável por uma disciplina pode corrigir as provas de outros professores que ministram a mesma disciplina.
Nessa versão inicial do projeto a feature de atividade/prova fixou o número de questões em 10, mas futuramente será possível criar atividades/provas com um número variável de questões. Também será adicionado a possibilidade de criar atividades/provas com questões de múltipla escolha, de modo que o professor poderá criar atividades/provas com questões de múltipla escolha e questões abertas.
Outro ponto importante do nosso projeto é que o Moodle Website Incrível não tem uso limitado apenas para alunos e professores, mas sim outros tipos de entidades, como por exemplo, empresas que podem utilizar a plataforma para criar atividades/provas para seus funcionários, de modo que a empresa pode avaliar o desempenho de seus funcionários. Também é possível que a empresa crie atividades/provas para seus clientes, de modo que a empresa pode avaliar o desempenho de seus clientes, ou qualquer outro tipo de entidade que tenha interesse em utilizar a plataforma.

Sobre a parte de Desenvolvimento
O projeto foi desenvolvido utilizando a linguagem de programação Java, e o framework Java Spring Boot. O banco de dados utilizado foi o MariaDb, e o sistema de controle de versão utilizado foi o Git. O projeto foi desenvolvido utilizando a metodologia ágil Scrum, e o projeto foi desenvolvido utilizando a ferramenta de gerenciamento de projetos Trello. O projeto foi desenvolvido utilizando a IDE Eclipse, e o projeto foi idealizado para ser hospedado na plataforma Heroku, porém com o fim da gratuidade da plataforma Heroku, e dado que essa versão inicial do projetos visa apenas a avaliação do projeto, o projeto funciona localmente, e não está hospedado na plataforma Heroku.
